¿Qué es HTML?
Estructura básica de un documento HTML (etiquetas <html>, <head>, <body>).

Listas:

Listas ordenadas <ol> y listas desordenadas <ul>.

Etiqueta de enlace <a>.

Imágenes:

Etiqueta de imagen <img>.
Atributo src para definir la fuente de la imagen.
Atributo alt para proporcionar un texto alternativo.

Elementos de entrada <input> para campos de texto y otros tipos de entrada.
Elemento de área de texto <textarea>.
Elemento de selección <select> y opciones <option>.
Elementos Semánticos:

Etiquetas semánticas como <header>, <nav>, <main>, <section>, <article>, <footer>.
Tablas:

Etiqueta de tabla <table>.
Filas <tr>, encabezados de columna <th>, celdas <td>.
Combinación de celdas con colspan y rowspan.
